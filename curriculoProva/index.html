Sobre mim

Sou estudante do ensino técnico integrado em Desenvolvimento de Sistemas pelo COLTEC/UFMG. Meu interesse por tecnologia surgiu no 9º ano, ao participar de uma oficina com Arduino, e desde então venho explorando áreas como programação e eletrônica. Fui bolsista de Iniciação Científica Júnior pela FAPEMIG, atuando no CEMTEC/UFMG com gerenciamento de dados da Estação Solarimétrica da Usina Tesla. Desenvolvi um dashboard com PostgreSQL e Grafana, posteriormente migrado para o Looker Studio para ampliar a acessibilidade e integração dos dados. Atualmente, participo do desenvolvimento de um aplicativo web para o projeto educacional Coltrekking, utilizando o Firebase Console para facilitar a organização de trekkings estudantis, incluindo controle de participantes e registro de atividades. Sou entusiasta de tecnologia, adoro aprender e estou sempre em busca de novos desafios e oportunidades de crescimento.
